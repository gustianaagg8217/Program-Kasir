## TROUBLESHOOTING CHART

Kalau problema ini:

### "Command prompt error"
```
→ Check: cd d:\Program_Kasir benar apa tidak
→ Cek: apakah file gui_main.py ada di folder itu
→ Cek: apakah punya Python installed (python --version)
```

### "Program terbuka lambat (> 5 detik)"
```
→ Disable recommendations (Langkah 2)
→ Check: database file (dir trades.db) berapa MB?
→ Try: close program lain yang berat
```

### "Program crash saat buka"
```
→ Run: python test_imports.py
→ Lihat: ada error apa di output
→ Report: error message tersebut
```

### "Emoji atau character error"
```
→ Update: gunakan file gui_main.py terbaru
→ Ensure: file sudah di-optimize (no emoji)
→ Try: jalankan lagi
```
